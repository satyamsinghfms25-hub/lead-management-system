## 🚀 Option A: Deploy Online (Recommended)

**Fastest way to get a live website on the internet:**

1. **Install Git**: https://git-scm.com/
2. **Create GitHub account**: https://github.com/signup  
3. **Open PowerShell in your project folder** and run:
   ```powershell
   git init
   git add .
   git commit -m "Lead Management System"
   git remote add origin https://github.com/YOUR_USERNAME/lead-management-system.git
   git push -u origin main
   ```
4. **Go to Vercel**: https://vercel.com/signup
5. **Sign in with GitHub** → **Import project** → **Deploy**

**Your site is LIVE!** 🌐 Get a URL like: `https://lead-system.vercel.app`

---

## 🏠 Option B: Run Locally

**For testing before deploying:**

1. **Install Node.js**: https://nodejs.org/ (LTS version)
2. **Open PowerShell in project folder**
3. **Run**:
   ```powershell
   npm install
   npm start
   ```
4. **Visit**: http://localhost:3000
